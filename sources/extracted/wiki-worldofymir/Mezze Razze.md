# Mezze Razze

- **Source:** https://worldofymir.fandom.com/wiki/Mezze_Razze
- **Wiki:** World of Yimir (worldofymir.fandom.com)
- **Last edited:** 2010-10-30T13:26:51Z

---

Queste non sono razze vere e proprie, bensì il risultato di un incrocio fra membri di razze diverse.

Nella lista qui sotto sono segnati gli incroci possibili fra le varie razze che popolano Ymir, per ogni mezza razza risultante viene dato il nome tecnico, il nome comune (come i meno dotti chiamano gli individui di quella specie, spesso il termine è dispregiativo) fra parentesi tonde e, fra parentesi quadre, se la razza è Sterile o in grado di figliare.

Umano + Elfo -> Yafysh (Mezzelfo) [S]

Umano + Orco -> Ghzaak (Mezzorco) [F]
Umano + Nano -> Gurbad (Mezzo-Nano) [S]
Umano + Rahab -> Uomo-Pesce [S]
Umano + Tanoth -> An-Tanoth (Mezzo Tanoth) [S]
Tanoth + Rahab -> Sibert [S]
Orco + Elfo -> Ghynr (Immondo) [S]
Orco + Tanoth -> Gha-Tanoth (Orco-Tanoth) [F]

Goblin + Nano -> Dwergar (Nano Immondo) [F]
Goblin + Orco -> Hobgoblin [F]
Goblin + Tal-Tanoth -> Lash-Tanoth [S]
Coboldo + Fata -> Gremlin [S]
Coboldo + Gnomo -> Pishacha (Gnomo Famelico) [S]
Troll + Tanoth -> Ugra-Tanoth (Tanoth Oscuro) [F]
Sohleuhgir + Gratar -> Kappa [F]

Umano + Fata -> Changeling [S]
