# canon/ — Structured world knowledge

This is the extracted, organized version of everything in `sources/`. Each file
covers one domain of the world. Facts here carry a tag (`[CANON]`, `[INFERRED]`,
`[INVENTED]`, `[CONFLICT]` — see the root `README.md`) and, where possible, a
pointer back to the source they came from.

These files start as skeletons and fill in as material arrives. They are the
bridge between the raw pile in `sources/` and the polished chapters in `manual/`.

## Topic files

| File | Covers |
|------|--------|
| `cosmology.md` | Creation myth, planes/realms, the nature of reality |
| `geography.md` | Continents, regions, cities, terrain, climate, maps |
| `history.md` | Timeline, ages, wars, cataclysms, major events |
| `peoples.md` | Races, cultures, languages, everyday life |
| `factions.md` | Nations, guilds, orders, cults, criminal powers |
| `pantheon.md` | Gods, spirits, religions, worship |
| `magic.md` | How magic works in-world; rules, limits, traditions |
| `bestiary.md` | Creatures, monsters, notable beasts |
| `characters.md` | Named NPCs and famous player characters |
| `calendar.md` | Calendar, cosmological cycles, festivals |
| `items.md` | Named weapons, relics, the in-world library (beyond magic.md's compact canon) |

New topic files are added whenever the material demands one.
